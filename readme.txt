WooCommerce Product Comparison and Specification gives you the best way to compare products based on WooCommerce attributes.

== Description ==

Product Comparison is a key feature to attract the customers to choose between the similar products which has attribute differences.

Plugin is compatible with WooCommerce 3.0+ and more. 

== Features of the plugin ==

*   <strong> Free version supports only WooCommerce attributes </strong>
*   Show a Comparison tab under product tabs on your single product page.
*   You can compare the all products that have the product categories of the current product.
*   You can compare the products by WooCommerce attributes.
*   You can customize the comparison table which will look very smooth like displaying the price, link to product and the size of thumbnail.

For complete list of settings documentation, please visit <a rel="nofollow" href="https://stepqueue.com/documentation/woocommerce-product-comparison-and-specification-setup/">How To Setup WooCommerce Product Comparison And Specification?</a>

== Premium version Features ==

<ul>
    <li> Supports Custom Specification Attributes.</li>
    <li> Specification tab which allows you to describe about your product specification precisely.</li>
    <li> Comparison also can be done by specification attributes or WooCommerce attributes.</li>
    <li> Instead of comparing all the product from the product categories you can compare only specified products which you configure.</li>
    <li> From the comparison product table customer can add the required product directly to cart with out going to product page.</li>
    <li> Premium support!</li>
</ul>

For complete list of features and details, please visit <a rel="nofollow" href="https://stepqueue.com/plugins/woocommerce-product-comparison-and-specification/">WooCommerce Product Comparison and Specification Pro</a>

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'WooCommerce Product Comparison and Specification'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `woo-product-comparison-and-specification.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `woo-product-comparison-and-specification.zip`
2. Extract the `woo-product-comparison-and-specification` directory to your computer
3. Upload the `woo-product-comparison-and-specification` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
